# AddonInstanceType

The instanceType of the add-on. Single instance add-ons can be added to subscription only once while add-ons with multiple type can be added more then once.

## Enum

* `SINGLE` (value: `'single'`)

* `MULTIPLE` (value: `'multiple'`)
