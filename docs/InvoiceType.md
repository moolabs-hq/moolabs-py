# InvoiceType

InvoiceType represents the type of invoice.  The type of invoice determines the purpose of the invoice and how it should be handled.

## Enum

* `STANDARD` (value: `'standard'`)

* `CREDIT_NOTE` (value: `'credit_note'`)
